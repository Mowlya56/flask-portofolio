# Flask Portfolio Website

This is a simple **Flask-based Personal Portfolio Website** built as part of the Python Developer Internship (Task 6).

## 🧠 Key Features
- Flask Routing and Templates
- Jinja2 Templating
- Contact Form using POST method
- Static CSS Styling

## ⚙️ Tech Stack
- Python
- Flask
- HTML5, CSS3
